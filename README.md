📦 Installation Guide

📱 Termux (Android)

```bash
pkg update && pkg upgrade -y
pkg install git curl -y
git clone https://github.com/shahid2005a/DGTLMAN.git
cd DGTLMAN
bash DGTLIP
```

🐧 Linux (Ubuntu/Debian/Kali)

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install git curl -y
git clone https://github.com/shahid2005a/DGTLMAN.git
cd DGTLMAN
sudo bash DGTLIP
```

🔵 Linux (Fedora)

```bash
sudo dnf update -y
sudo dnf install git curl -y
git clone https://github.com/shahid2005a/DGTLMAN.git
cd DGTLMAN
sudo bash DGTLIP
```

🟡 Linux (Arch)

```bash
sudo pacman -Syu --noconfirm
sudo pacman -S git curl --noconfirm
git clone https://github.com/shahid2005a/DGTLMAN.git
cd DGTLMAN
sudo bash DGTLIP
```

⚡ Single Command Installation

Termux:

```bash
pkg update ; pkg install git curl -y ; git clone https://github.com/shahid2005a/DGTLMAN.git ; cd DGTLMAN ; bash DGTLIP
```

Linux:

```bash
sudo apt update && sudo apt install git curl -y && git clone https://github.com/shahid2005a/DGTLMAN.git && cd DGTLMAN && sudo bash DGTLIP
```
